# sCrappi
Sistema  Control Remoto de Acceso y Permanencia de Personal inSitu 🚀

Ejecutar Back 
crear la base de datos en postgres con el nombre que está en el archivo .yml
$env:DB_USER="postgres"; $env:DB_PASSWORD="tucontraseña"; ./mvnw spring-boot:run

Instalar node_modules
npm install
ejecutar Front
ng s
